# Tauri Application Icons

Place your application icons here:

- 32x32.png - 32x32 icon
- 128x128.png - 128x128 icon
- 128x128@2x.png - 256x256 icon (for retina displays)
- icon.icns - macOS icon
- icon.ico - Windows icon

You can generate these from a 1024x1024 source image using:
- Tauri's `tauri icon` command
- Or online tools like favicon.io
